## Задание 1.
### Текст задачи
Разработать консольное приложение с дружественным интерфейсом с возможностью выбора
заданий для работы с «базой данных (БД)», хранящейся в бинарном файле. Перечень полей
(минимум 5), достаточно полно характеризующих заданную в варианте предметную область,
предложить самостоятельно (постараться отразить в перечне полей такие, которые требуют разных
типов данных). Приложение должно выполнять следующие функции:
1. Чтение базы данных из бинарного файла.
2. Просмотр базы данных.
3. Удаление элементов (по ключу).
4. Добавление элементов.
5. Реализация 4 запросов (формулировки запросов придумать самостоятельно). 2 запроса
должны возвращать перечень, 2 запроса одно значение.
В классе должны присутствовать свойства, конструкторы, перегруженный метод ToString(). Весь
функционал приложения реализовать в виде методов вспомогательного класса с помощью LINQзапросов.
### Алгоритм решения
1. Базовый класс Shop содержит 5 полей: id товара, название товара, категория к которой он принадлежит, цену и количество. Реализованы конструктора по умолчанию, с параметрами и конструктор копирования. Для каждого поля созданые свои свойства, в которых реализованы проверки (id > 0, price > 0, count > 0, name и category не пустые). Также переопределен метод ToString.
2. Для работы с бинарными файлами: метод Fill, который заполняет файл по умолчанию, метод PrintBinaryFile, которые выводит содержимое бинарного файла.Так как работы с бинарными файлами напрямую неудобна, используется список. Данные из файла загружаются в список с помощью метода Load. После обработки данные сохраняются обратно в файле методом Save.
3. Реализован метод добавления элемента: данные вводятся пользователем, добавляются в список и сохраняются в файл. Реализован метод удаления по ид: поиск этого элемента, удаление из списка, сохранение в файл.
4. Реализованы 4 запроса: товары дороже заданной цены (Where), товары с малым количеством (Where), средняя цена товара (Average), самый дорогой товар (OrderByDescending + First).
5. В main данные тестируются и выводятся на экран.
